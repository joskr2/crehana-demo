
# Crehana demo

An application that allows you to find information about the different
countries of the world

## Installation

Clone this repository on your local machine 


## Usage

```javascript
npm install

npm start 
```

## Tech stack
Frontend: 

1. React(Typescript)
2. Apollo(GraphQL)
3. Recoil(State Managment)
4. TaildwindCSS(styles)
